# Label Chest

**Label Chest** is a lightweight quality-of-life (QoL) mod for Stardew Valley that helps you keep your farm organized. It allows you to assign custom text labels to any chest, which are then displayed as floating text in the game world.

---

## ğŸŒŸ Features

*   **Integrated UI**: Adds a "Set Label" button directly inside the chest inventory menu for quick access.
*   **In-World Labels**: Your custom labels float neatly above your chests, using a clean, vanilla-friendly font style.
*   **Smart Text Wrapping**: Long names are automatically wrapped into multiple lines to ensure they remain readable on your screen.
*   **Safe Data Storage**: Labels are saved using the game's native `modData` system. This means your labels are tied to your save file and won't be lost if you update the mod.
*   **Multiplayer Compatible**: Works perfectly in co-op sessions!

---

## ğŸ® How to Use

1.  **Open a Chest**: Interact with any chest to open its inventory.
2.  **Click the Button**: Look for the **"Set Label"** button located just above the chest inventory slots.
3.  **Enter Text**: Type your desired name in the pop-up naming menu.
4.  **Confirm**: Click the **Checkmark (OK)** button. The label will immediately appear above the chest in the world.
5.  **Remove/Edit**: To change a label, click the button again. To remove it entirely, clear the text box and click OK.
